# Dataset Dictionary

## Overview

This document describes variables that can be included in the pilot dataset.


| Variable Name | Description | Type | Example |
|--------------|-------------|------|---------|
| encounter_id | Unique encounter identifier | String | ENC001 |
| facility_id | Facility identifier | String | FAC01 |
| visit_date | Date of encounter | Date | 2026-07-06 |
| age | Patient age | Integer | 18 |
| sex | Biological sex | Categorical | Female |
| chief_complaint | Primary reason for visit | Text | Fever |
| diagnosis | Clinical diagnosis | Text | Sickle Cell |
| treatment | Treatment provided | Text | Hydroxyurea |
| follow_up | Follow-up planned | Yes/No | Yes |


## Missing Data Codes

NA = Not Available

UNK = Unknown

NR = Not Recorded
