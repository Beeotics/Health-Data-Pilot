# Pilot Protocol

## Title

Lightweight clinical data Capture pilot in Kinshasa to identify missing populations in global health datasets


# Objective

To investigate where routine clinical information becomes invisible within healthcare workflows and to evaluate the feasibility of a lightweight structured data capture approach.


# Study Duration

8–12 Weeks


# Study Setting

Selected healthcare facilities in Kinshasa, Democratic Republic of the Congo.

Facility selection will be based on:

* Willingness to participate
* Feasibility of access
* Diversity of clinical workflows


# Study Activities

## Activity 1: Workflow Mapping

Document existing patient information flows.

Outputs:

* Workflow diagrams
* Process descriptions
* Data loss assessment


## Activity 2: Staff Interviews

Conduct semi-structured interviews with:

* Physicians
* Nurses
* Administrative staff
* Data personnel

Target:

10–20 interviews


## Activity 3: Structured Data Capture Pilot

Implement a lightweight capture template.

Potential fields:

* Patient demographics
* Visit date
* Chief complaint
* Diagnosis
* Treatment
* Follow-up status

Target:

300–1000 encounters (depending on feasibility)


## Activity 4: Analysis

Assess:

* Completeness
* Consistency
* Feasibility
* Operational burden


# Data Management

All collected information will be:

* De-identified
* Stored securely
* Access-controlled
* Used only for project purposes

No personally identifiable information will be included in public outputs.


# Deliverables

1. Data flow map
2. Field report
3. Structured dataset sample
4. Data capture template
5. Recommendations document


# Success Criteria

* Workflow mapping completed
* Interviews completed
* Pilot dataset generated
* Report delivered within project timeline
